# TestWork

## Запуск проекта

Нужны установленные **Docker**, **Docker Compose** и **Make**.

Из корня репозитория выполните:

```bash
make init
```

Команда поднимает окружение в Docker: собирает образы, создаёт `.env` из примера при необходимости, ставит зависимости PHP и Node, генерирует ключ приложения, выполняет `migrate:fresh --seed`.

## Запуск для разработки

После успешного `make init` проект можно запустить командой:

```bash
make up-dev
```

Поднимутся контейнеры и фронтенд-сервер разработки (Vite).  `http://localhost` (порты заданы в `docker-compose.yml`).
